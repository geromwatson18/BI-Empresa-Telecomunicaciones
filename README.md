# Proyecto BI – Empresa de Telecomunicaciones

## Descripción del caso
Este proyecto implementa una solución completa de Inteligencia de Negocios para una empresa de telecomunicaciones costarricense, enfocada en el análisis de ventas, clientes, incidencias técnicas y churn.

## Arquitectura de la solución
Base de datos operacional (SQL Server)
→ Proceso ETL (SQL)
→ Data Warehouse (modelo dimensional)
→ Visualización analítica (Power BI)

## Herramientas utilizadas
- SQL Server
- Scripts SQL para ETL
- Power BI Desktop
- GitHub

3. Ejecutar el script ETL
